# Password Manager (Tkinter + JSON)

This Python project is an advanced Password Manager built using Tkinter. It allows users to generate strong passwords, save website credentials in a structured JSON file, and search stored passwords. It includes data validation, file handling, and a user-friendly GUI for secure credential management.

---

## 📌 Features

- Strong random password generator
- Automatic clipboard copy
- Save credentials in JSON format
- Search saved passwords by website
- Error handling for missing files/data
- Simple and interactive GUI

---

## 🛠 Technologies Used

- Python 3
- Tkinter GUI Library
- JSON module
- Random module
- Pyperclip module
- Messagebox (Tkinter)
